Displays the pending comments (comments requiring approval) on a separate page as an RSS feed. 

== Description ==

<p>Feed for pending comments is a simple Wordpress plugin that generates a RSS feed for pending comments on your website or blog. You can access the pending comments feed by visiting yoursite.com/pencom-feed.</p>
<p>This plugin is actually meant to go together with a recipe from IFTTT.com. By using this plugin and the site IFTTT.com, you'll be able to receive SMS everytime someone comments on your site. You can read the tutorial/guide on how to do it here : <a href="http://www.thelacunablog.com/?p=924" alt="Receive SMS everytime you get a new comment" title="The Lacuna Blog">http://www.thelacunablog.com/?p=924</a></p>
<p>We post computer tips, guides and how to's. Visit us here: <a href="http://www.thelacunablog.com/" alt="The Lacuna Blog" title="The Lacuna Blog">www.thelacunablog.com </a></p>


== Installation ==

Here is what you need to do: 

1. Install plugin OR unzip the downloaded file to /wp-content/plugins/
1. Go to your plugin settings and enable the plugin.
1. Go to yoursite.com/pencom-feed to access the RSS feed.

== Frequently Asked Questions ==

= Where is the feed located? =

You can access the feed by going to yoursite.com/pencom-feed.

= Can I use custom page? =

No, the ability to use custom page for showing RSS feed is not available right now.
